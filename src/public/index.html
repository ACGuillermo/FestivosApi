Introduction

Welcome to Festivos API! This documentation should help you familiarise yourself with the resources available and how to consume them with HTTP requests.

Getting started

Open up a terminal and use curl to make an API request for a resource. In the next example we are trying to get all the holidays of Barcelona:

curl festivosapi.dev/api/municipios/barcelona/
